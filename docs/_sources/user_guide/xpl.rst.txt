File explorer overview
======================
在 ``Mapdl`` 类下有一个专用于 XPL 的特定属性。它基于 APDL Math ``*XPL`` 命令，可用于探索由 APDL 生成的二进制文件。

所有 APDL 文件的组织方式相同，都是树形或记录形。例如，二进制结果文件的结构如图 :ref:`ref_rst_structure` 所示。

.. _ref_rst_structure:

.. figure:: ../images/rst_structure.png
    :width: 500pt

    APDL 结果文件的结构

您可以在 :ref:`ref_xpl_example` 中找到 APDL 文件资源管理器的详细示例。

更多信息，请参阅 :ref:`ref_xpl_api`。
