Download example files
======================

每个示例都应包含运行该示例所需的全部资源。但在某些情况下，还需要外部文件。每个示例页面都有指向这些文件的链接。
这些链接指向以下 GitHub 代码库，您可以在其中找到所有链接：

`GitHub Example Data Repository <example_data_repo_>`_

如果发现链接丢失或断开，请在 GitHub 上打开一个问题（ `PyMAPDL Issues <pymapdl_issues_>`_ ）。
